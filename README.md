# PokerTrainer

Interactive poker training app — pot odds calculator, preflop trainer, and range charts.

## Stack

- Next.js 16, React 19, TypeScript
- Tailwind CSS v4, shadcn/ui
- Bilingual: English + Ukrainian

## Development

```bash
npm install
npm run dev
```

## Deploy

Deployed on Railway. Build command: `npm run build`, start command: `npm start`.
